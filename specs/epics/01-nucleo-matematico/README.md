# Épico 1 — Núcleo Matemático e Motor de Harmonia

> **Pacote:** `@hearmony/core`

## Objetivo

Implementar o motor matemático que representa relações harmônicas como grafos ponderados, utilizando matrizes de adjacência para calcular probabilidades de progressão de acordes.

## Specs deste Épico

| Spec | Título | Status |
|------|--------|--------|
| SPEC-1.01 | Matriz de Adjacência Harmônica | 📝 DRAFT |
| SPEC-1.02 | Validador Cromático | 📝 DRAFT |
| SPEC-1.03 | Engine Geométrica (Fretboard) | 📝 DRAFT |

## Escopo

- Representação de acordes e notas como nós de um grafo
- Cálculo de pesos de transição entre acordes
- Validação cromática de intervalos e formações
- Engine geométrica para mapeamento de posições no braço
